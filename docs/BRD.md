## 1. Цель продукта

Нативный виджет для macOS, показывающий актуальные поисковые тренды
(Google Trends) для России и Мира прямо на рабочем столе / в панели
виджетов, без необходимости открывать браузер.

## 2. Заинтересованные стороны

| Роль | Кто | Интерес |
|---|---|---|
| Заказчик / пользователь | Владелец проекта | Быстрый доступ к трендам |
| Разработка | Claude Code | Реализация полного цикла |

## 3. Область (Scope)

### Входит в v1.0
- **Форм-фактор:** виджет WidgetKit (Notification Center / рабочий стол
  macOS) + минимальное приложение-контейнер (обязательно для установки
  виджета, содержит инструкцию и предпросмотр трендов).
- **Источник данных:** внутренний API Google Trends «Trending Now»
  (`trends.google.com/_/TrendsUi/data/batchexecute`, метод `i0OFE`):
  отдаёт запрос, объём поиска, процент роста и время старта тренда.
  Без ключей и авторизации. Аварийный фолбэк — официальный RSS-фид
  (топ за день, без процента роста).
- **Регионы (конфигурируемые):** Россия (`RU`), США (`US`),
  Европа (агрегация DE/FR/IT/ES/PL/NL), Мир (агрегация
  US/GB/DE/FR/BR/IN/JP/RU). У Google нет гео «весь мир» и «Европа»,
  агрегация = дедупликация + максимум метрики по регионам —
  зафиксировано с заказчиком как допустимое приближение.
- **Окно времени (конфигурируемое):** час / день / неделя.
  «Месяц» недоступен в источнике (максимум 7 дней) — заменён
  на «неделю» по согласованию.
- **Разделы (конфигурируемые):** «Топ» — сортировка по объёму поиска;
  «Райзинг» — по проценту роста популярности.
- **Конфигурация виджета:** регион, окно, раздел и интервал обновления
  выбираются через штатное меню «Редактировать виджет»
  (AppIntent-конфигурация); можно держать несколько виджетов с разными
  настройками одновременно.
- **Размеры виджета:** small (топ-3), medium (топ-5), large (топ-10)
  с показателем объёма («500+», «100K+») и ростом («+900%») для райзинга.
- **Обновление:** настраиваемый интервал 15 / 30 / 60 минут
  (по умолчанию 30); при смене настроек виджет обновляется мгновенно
  из кэша с фоновой дозагрузкой; при недоступности сети — показ
  последних успешно загруженных данных с пометкой времени.
- **Взаимодействие:** клик по запросу открывает страницу тренда в
  браузере.

### Не входит в v1.0 (фаза 2+)
- Яндекс как источник (Вордстат-API требует OAuth; публичного API
  трендов нет) — отложено по решению заказчика.
- Публикация в Mac App Store, подпись Developer ID и нотаризация
  (нет аккаунта Apple Developer Program).
- iOS/iPadOS-версии, локализации кроме русской.
- Графики динамики, история трендов, уведомления.

## 4. Нефункциональные требования

| Категория | Требование |
|---|---|
| Платформа | macOS 14.0+ (Sonoma), Apple Silicon и Intel (universal) |
| Язык/стек | Swift 5.10+/SwiftUI/WidgetKit, без сторонних runtime-зависимостей |
| Производительность | Ответ фида парсится < 100 мс; виджет не будит систему чаще расписания WidgetKit |
| Надёжность | Грациозная деградация: кэш последних данных, явное состояние «нет сети/данных» |
| Безопасность | Только HTTPS (ATS по умолчанию), App Sandbox, единственный entitlement — исходящая сеть; секретов в коде нет |
| Приватность | Приложение не собирает и не отправляет никаких данных пользователя |
| Дистрибуция | GitHub Releases: DMG без подписи (ad-hoc), инструкция по обходу Gatekeeper для личного использования |
| CI/CD | GitHub Actions: сборка, юнит-тесты, линт на каждый PR/push; релизный DMG по git-тегу |

## 5. Критерии приёмки v1.0

1. Виджет добавляется в панель виджетов macOS и показывает актуальные
   тренды для выбранного региона (Россия / Мир).
2. Регион переключается через «Редактировать виджет».
3. Все три размера виджета отображаются корректно (светлая/тёмная тема).
4. При отключённой сети виджет показывает последние данные и время
   их получения, не «падает» в пустое состояние.
5. Клик по строке тренда открывает соответствующую страницу Google Trends.
6. CI-пайплайн зелёный: тесты парсера/агрегатора + SwiftLint проходят.
7. ~~По тегу `v*` автоматически публикуется GitHub Release с DMG.~~
   Снято 2026-07-23: ad-hoc DMG несовместим с конфигурацией виджета
   (AppIntents требует Team ID подписи) — дистрибуция только исходниками.
